# Support Guide

Thank you for using `advanced_device_info`! We want to ensure you have the best possible experience when using this library. Here is how you can get help or report issues.

## 🐛 Reporting Bugs & Requesting Features

If you encounter a bug, unexpected behavior, or have an idea for a new feature:
1. Check the existing issues in our issue tracker to see if it has already been reported.
2. If it hasn't, open a new issue detailing:
   * The version of the package you are using.
   * Target platform (Android/iOS) and OS version.
   * Clear steps to reproduce the issue.
   * Code snippets or a minimal reproducing project if possible.

## 💬 Questions & Discussions

For general questions, implementation advice, or discussions on best practices, please open a discussion in our community discussions page or ask on Stack Overflow using the `flutter` and `dart` tags.
