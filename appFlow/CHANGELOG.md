# Changelog

## 1.0.0
- Initial project structure.
- Completed Phase 1 Core Device Information:
  - Added Device Info (Name, Model, Brand, Manufacturer, isEmulator status, and platform-specific build details).
  - Added Storage Info (Total, Free, Used, and Usage percentage).
  - Added Memory / RAM Info (Total, Available, Used, and Usage percentage).
  - Added CPU Info (Architecture and core count).
  - Added Screen Info (Physical width, height, density scale, and refresh rate).
  - Added unified System Info API to fetch all metrics in a single MethodChannel call.
